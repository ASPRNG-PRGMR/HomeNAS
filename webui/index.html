Detection rules run automatically after every event batch. Each rule fires an alert when its condition is met and respects a cooldown period to avoid flooding the alert table during a sustained attack.
Brute Force (BF) IP-keyed — triggered by login failure patterns from a single source IP
RuleConditionWindowSeverity
BF-001 5 or more login failures from the same IP 60s Low
BF-002 10 or more login failures from the same IP 60s Medium
BF-003 20 or more login failures from the same IP 60s High
BF-004 50 or more failures in 5 min, or 100 or more in 15 min from same IP 5–15 min Critical
BF-005 10 or more failures followed by a successful login from the same IP — attacker may have gained access 10 min High
BF-006 Same username targeted from 3 or more distinct IPs — distributed password spraying 5 min Medium
Mass Delete (MD) Actor-keyed — triggered by deletion patterns from an authenticated user. Directory deletes count the same as file deletes.
RuleConditionWindowSeverity
MD-001 20 or more deletes by the same user 60s Low
MD-002 50 or more deletes by the same user 60s Medium
MD-003 100 or more deletes by the same user 5 min High
MD-004 500 or more deletes by the same user — possible ransomware wipe 10 min Critical
MD-005 Login followed immediately by 50 or more deletes — possible compromised account 5 min High
MD-006 Any directory deletion — fires on every occurrence, no cooldown Per event Medium
Path Traversal (PT) IP-keyed — triggered when requests attempt to access paths outside the storage root (e.g. ../../etc/passwd). One or two occurrences may be a misconfigured client; a burst indicates active probing.
RuleConditionWindowSeverity
PT-001 5 or more path traversal attempts from the same IP 60s Medium
PT-002 20 or more path traversal attempts from the same IP — sustained directory probing 5 min High
Data Exfiltration (DX) Actor-keyed — triggered by abnormal download volume from an authenticated user. Keyed on actor, not IP, so rotating exit nodes does not evade detection.
RuleConditionWindowSeverity
DX-001 50 or more file downloads by the same user 5 min Medium
DX-002 200 or more file downloads by the same user 10 min High
DX-003 Login followed immediately by 30 or more downloads — possible stolen credential being used to exfiltrate data 5 min High
